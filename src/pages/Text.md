
# Text

Responsive typographic component.
Extends [Box](/Box).

```.jsx
<Text
  fontSize={[ 3, 4, 5 ]}
  fontWeight='bold'
  color='magenta'>
  Text
</Text>
```

## Props

In addition to the base [Box](/Box) props,
Text includes the following style props:

- `fontFamily`
- `fontWeight`
- `textAlign`
- `lineHeight`
- `letterSpacing`
