
# Image

Image with max-width and style props. Extends [Box](/Box)

```.jsx
<Image
  width={[ 1, 1, 1/2 ]}
  src='https://source.unsplash.com/random/1280x720'
  borderRadius={8}
/>
```

## Props

In addition to the base [Box](/Box) props, Image includes the following style props.

- `height`
- `borderRadius`
